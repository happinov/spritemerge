# spritemerge

## Description
A tool that combines a bunch of png files into a single png which is a combination of all source images placed in a single row.
The resulting image can then be used to create CSS animations.

Note that resulting PNG image isn't optimized in any particular way so you might want to apply some processing like [optipng](http://optipng.sourceforge.net/) or [pngcrush](http://pngcrush.com/).

## Installation

```
$ npm install -g https://github.com/happinov/spritemerge.git
```

## Usage

```
$ spritemerge <source folder> <target image>
```
